# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Guía para los clientes de Azure Data Lake Storage Gen1

La versión preliminar de Azure Data Lake Storage Gen2 ofrece la funcionalidad de almacenamiento de blob nativo, incluidas capas de datos, administración de ciclo de vida y seguridad de red mejorada, además del sistema de archivos compatibles con Hadoop que está disponible en Data Lake Storage Gen1.

Si usted ya es cliente de Data Lake Storage Gen1, Microsoft le anima a evaluar todas las capacidades nuevas y mejoradas de Data Lake Storage Gen2 y a proporcionar comentarios [participando en la versión preliminar](https://aka.ms/adlsgen2signup). Data Lake Storage Gen1 se seguirá admitiendo. Data Lake Storage Gen1 y Gen2 pueden coexistir, por lo que los recursos y aplicaciones existentes que aprovechan Data Lake Storage Gen1 continuarán funcionando sin actualizaciones. Puede desarrollar un plan de migración para Data Lake Storage Gen2 según le convenga.

Si en la actualidad está evaluando Data Lake Storage Gen1, Microsoft le anima a participar en la versión preliminar de Gen2 y a evaluar las capacidades nuevas y mejoradas que ofrece Data Lake Storage Gen2. De forma similar, si ya es cliente de Blob Storage, Microsoft le anima a evaluar si un sistema de archivos compatible con Hadoop es adecuado para sus necesidades de análisis.
