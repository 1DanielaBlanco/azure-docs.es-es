# Administración de datos: Información sobre NoSQL en Microsoft Azure

**Resumen:** Azure ofrece una amplia gama de opciones para las cargas de trabajo NoSQL. Sin embargo, pero puede resultar complicado determinar las diferencias entre todas las opciones disponibles. Este documento describe las tecnologías NoSQL de Azure y explica las diferencias de administración que existen entre los modelos relacionales de la administración de los datos.

Desde hace tiempo, la tecnología relacional es el método dominante para trabajar con datos. Sin embargo, al trabajar con grandes cantidades de distintos tipos de datos, como sucede cada vez más hoy en día, se plantean nuevos desafíos para la administración de estos datos. La tecnología relacional a veces es la mejor opción para la administración de estos datos y a veces no. En esta guía se enumeran las opciones y se explican sus características y los motivos para elegir una u otra.
