# <a name="develop-secure-applications-on-azure"></a>Desarrollo de aplicaciones seguras en Azure
## <a name="abstract"></a>Descripción breve
Este artículo es una guía general que incluye los controles y las preguntas de seguridad que se deben considerar en cada fase del ciclo de vida de desarrollo de software al desarrollar aplicaciones para la nube. La implementación de estos conceptos antes de presentar su producto puede ayudarle a crear software más seguro. Las recomendaciones que se presentan en este artículo proceden de nuestra experiencia con la seguridad de Azure y de las experiencias de nuestros clientes.

Este artículo está pensado para ser un recurso para los diseñadores, los desarrolladores y los evaluadores de software de todos los niveles que compilan e implementan soluciones seguras de Azure.
