# Blog sobre seguridad y cumplimiento de Azure

El [blog del equipo de seguridad y cumplimiento de Azure](https://blogs.msdn.microsoft.com/azuresecurity/) es el lugar al que dirigirse para obtener la información técnica actualizada sobre la seguridad y el cumplimiento de Azure.

Además, busque el blog del equipo de seguridad y cumplimiento de Azure para obtener actualizaciones sobre temas técnicos para [Azure Security Center](https://azure.microsoft.com/documentation/services/security-center/).

Además de obtener información técnica, el blog del equipo de seguridad y cumplimiento de Azure le remite a los anuncios más recientes sobre la seguridad y el cumplimiento de Azure tal y como aparecen en el blog de Azure.

Asegúrese de consultar el blog del equipo de seguridad y cumplimiento de Azure para ver las novedades con todo lo relacionado con la seguridad de Azure.
