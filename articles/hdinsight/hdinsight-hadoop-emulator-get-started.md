# Introducción al ecosistema de Hadoop con un espacio aislado de Hadoop

Puede configurar un espacio aislado de Hadoop (desde Hortonworks) en una máquina virtual para aprender sobre el ecosistema de Hadoop. El espacio aislado proporciona un entorno de desarrollo local para comprender Hadoop, el sistema de archivos distribuido de Hadoop (HDFS), el envío de trabajos, etc. Para comenzar con un espacio aislado de Hadoop de Hortonworks, consulte [Hortonworks Sandbox](http://hortonworks.com/downloads/#sandbox) (Espacio aislado de Hortonworks) y concretamente la sección **Hortonworks Sandbox on a VM** (Espacio aislado de Hortonworks en una máquina virtual). Se recomienda explorar los tutoriales disponibles con el espacio aislado para comprender HDFS, cómo se envían trabajos a un clúster, cómo realizar el seguimiento de trabajos que se ejecutan en un clúster, etc.

Cuando se haya familiarizado con Hadoop, puede empezar a usarlo en Azure mediante la creación de un clúster de HDInsight. Para más información sobre cómo empezar, consulte [Tutorial de Hadoop: Introducción al uso de Hadoop en HDInsight basado en Linux](hdinsight-hadoop-linux-tutorial-get-started.md).
