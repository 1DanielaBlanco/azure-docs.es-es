# Servicios de dominio de Azure AD *(vista previa)*

## Regiones de Azure admitidas
Los Servicios de dominio de Azure Active Directory están disponibles en las siguientes regiones de Azure:

- Estados Unidos
  - Central EE. UU.:
  - Este de EE. UU.
  - Este de EE. UU. 2
  - Centro-Sur de EE. UU
  - Oeste de EE. UU.

- Asia
  - Asia oriental
  - Sudeste asiático
