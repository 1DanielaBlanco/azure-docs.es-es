# <a name="iot-solution-accelerators-documentation"></a>Documentación sobre los aceleradores de soluciones de IoT

Introducción rápida a los aceleradores de soluciones de IoT para compilar soluciones de este tipo. Agregue nuevos dispositivos y conéctese a los existentes utilizando los SDK de dispositivo para diversas plataformas, incluidas Linux, Windows y sistemas operativos en tiempo real. Escale fácilmente desde tan solo unos pocos sensores hasta millones de dispositivos conectados simultáneamente y confíe en la disponibilidad global de Azure, sea cual sea el tamaño del proyecto.
