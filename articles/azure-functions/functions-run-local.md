# <a name="how-to-code-and-test-azure-functions-locally"></a>Codificación y comprobación de las funciones de Azure en un entorno local
> [!NOTE]
> El desarrollo local de Azure Functions y la experiencia con herramientas actualmente se encuentran en vista previa, y la experiencia se mejorará significativamente antes del lanzamiento final. Para ejecutar el host de Azure Functions en un entorno local, lo más sencillo es usar la [CLI de Azure Functions](https://go.microsoft.com/fwlink/?linkid=832752) e instalarlo desde npm. Actualmente, solo se admite Windows. La documentación completa se incluirá próximamente. Para proporcionar comentarios o errores de archivos de la CLI de Azure Functions, envíe un problema en el repositorio de GitHub [azure webjobs sdk script](https://github.com/Azure/azure-webjobs-sdk-script).
