# Get-AzureRmFrontDoorFireWallPolicy

Obtener una directiva de WAF.

## SINTAXIS

```
Get-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> [-Name <String>]
 [-DefaultProfile <IAzureContextContainer>] [<CommonParameters>]
```

## Description
El cmdlet **Get AzureRmFrontDoorFireWallPolicy** obtiene la directiva de WAF en un grupo de recursos en la suscripción actual.

## EJEMPLOS

### Ejemplo 1: Obtener una directiva de WAF denominada $Name en $resourceGroup
```powershell
PS C:\> Get-AzureRmFrontDoorFireWallPolicy -Name $Name -ResourceGroupName $resourceGroup

PolicyEnabledState : Enabled
CustomRules        : {Rule1}

Tags               :

                     osoft.Network/frontdoorwebapplicationfirewallpolicies/{Name}

Type               :
```

Obtener una directiva de WAF denominada $Name en $resourceGroup
