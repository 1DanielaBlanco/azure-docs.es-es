# <a name="security-attributes-for-azure-key-vault"></a>Atributos de seguridad de Azure Key Vault

La seguridad está integrada en todos los aspectos de los servicios de Azure. En este artículo, se explican los atributos de seguridad integrados en Azure Key Vault. Un atributo de seguridad es una cualidad o característica de un servicio de Azure que ayuda a dicho servicio a prevenir y detectar vulnerabilidades de seguridad, así como a responder a ellas.

Los atributos de seguridad se clasifican en las siguientes categorías:
* Prevención
* Segmentación de red
* Detección
* Compatibilidad con la administración de identidades y acceso (IAM)
* Pista de auditoría
* Controles de acceso (si se utilizan)
* Administración de la configuración (si se utiliza)

En todas las categorías, especificamos si el atributo se utiliza o no (sí/no). En posible que, en algunos servicios, haya atributos que no sean aplicables, lo que se indica con N/A. También puede incluirse una nota o un vínculo para aportar más información sobre un atributo.
