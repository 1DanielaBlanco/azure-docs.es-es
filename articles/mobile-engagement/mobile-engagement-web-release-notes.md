#Notas de la versión del SDK web de Azure Mobile Engagement

##2\.0.1 (06/10/2016)

-   SDK deshabilitado en IE8 y IE9.
-   Detección fija de explorador web Opera.

##2\.0.0 (06/07/2016)

-   Versión inicial de Azure Mobile Engagement.
-   La configuración de appID se reemplaza por una configuración de cadena de conexión.
-   Se han añadido API para habilitar o deshabilitar al agente.
-   Mejoras de seguridad.
-   Uso de API nativas de JSON de los exploradores.
-   Se ha eliminado la API para enviar y recibir mensajes entre dispositivos.
-   Devoluciones de llamada relacionadas con la conexión XMPP en desuso quitadas. 
-   Esta versión no es compatible con la característica de cobertura.
