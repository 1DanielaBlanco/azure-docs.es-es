# <a name="pricing"></a>Precios

## <a name="compute-cost"></a>Costo de proceso
Azure Cloud Shell se ejecuta en una máquina proporcionada de forma gratuita, pero requiere un recurso compartido de archivos de Azure conectado para su uso. 

## <a name="storage-cost"></a>Costo del almacenamiento
Cloud Shell crea una imagen de 5 GB predeterminada en el recurso compartido de archivos para hacer persistente el directorio $HOME. Los recursos compartidos de archivos de Azure conllevan costos de forma regular.

Haga clic [aquí para obtener detalles sobre los costos de Azure Files](https://azure.microsoft.com/pricing/details/storage/files/).
