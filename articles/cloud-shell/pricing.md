# <a name="pricing"></a>Precios

## <a name="compute-cost"></a>Costo de proceso
Azure Cloud Shell se ejecuta en una máquina que Azure proporciona gratuitamente, pero requiere un recurso compartido de archivos de Azure para su uso. 

## <a name="storage-cost"></a>Costo del almacenamiento
Cloud Shell requiere que se monte un recurso compartido de archivos de Azure nuevo o existente para conservar archivos entre las sesiones. El almacenamiento incurre en costos normales.

Haga clic [aquí para obtener detalles sobre los costos de Azure Files](https://azure.microsoft.com/pricing/details/storage/files/).
