# <a name="xml-processing"></a>Procesamiento XML
Enterprise Integration Pack facilita la validación y el procesamiento de los documentos XML que intercambia con los asociados. Estas son los métodos de procesamiento de los mensajes XML mediante Logic Apps:  

* [Validación XML](logic-apps-enterprise-integration-xml-validation.md "Información sobre la validación de mensajes XML"): la validación XML permite validar un mensaje procedente de un punto de conexión de origen en un esquema concreto.
* [Transformación XML](../logic-apps/logic-apps-enterprise-integration-transform.md "Información sobre las asignaciones y transformaciones de mensajes XML"): la transformación XML permite convertir un mensaje XML basado en los requisitos de un punto de conexión de destino.   
* [Codificación y descodificación de archivos sin formato](logic-apps-enterprise-integration-flatfile.md "Información sobre la codificación y descodificación de archivos sin formato"): la codificación o descodificación de archivos sin formato permiten codificar o descodificar un archivo sin formato. 
* [XPath](https://msdn.microsoft.com/library/mt643789.aspx) : Permite enriquecer un mensaje y extraer propiedades específicas de él. Las propiedades extraídas se pueden utilizar para redirigir el mensaje a un destino o un punto de conexión intermediario.
