# <a name="xml-processing"></a>Procesamiento XML
Valide y procese los documentos XML que intercambia con asociados en escenarios de B2B más fácilmente mediante Enterprise Integration Pack con Azure Logic Apps. Estos son los métodos de procesamiento de mensajes XML:

* [Validación XML](logic-apps-enterprise-integration-xml-validation.md "Información sobre la validación de mensajes XML"): permite validar un mensaje procedente de un punto de conexión de origen en un esquema concreto.
* [Transformación XML](../logic-apps/logic-apps-enterprise-integration-transform.md "Información sobre las asignaciones y transformaciones de mensajes XML"): permite convertir un mensaje XML basado en los requisitos de un punto de conexión de destino.
* [Codificación y descodificación de archivos sin formato](logic-apps-enterprise-integration-flatfile.md "Información sobre la codificación y descodificación de archivos sin formato"): permiten codificar o descodificar un archivo sin formato. 
* [XPath](https://msdn.microsoft.com/library/mt643789.aspx): permite enriquecer un mensaje y extraer propiedades específicas de él. Las propiedades extraídas se pueden utilizar para redirigir el mensaje a un destino o un punto de conexión intermediario.
