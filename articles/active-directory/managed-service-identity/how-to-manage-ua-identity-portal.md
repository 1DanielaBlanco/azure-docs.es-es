# <a name="create-list-or-delete-a-user-assigned-managed-service-identity-msi-using-the-azure-portal"></a>Creación, enumeración o eliminación de una instancia de Managed Service Identity (MSI) asignada por el usuario mediante Azure Portal

[!INCLUDE[preview-notice](~/includes/active-directory-msi-preview-notice-ua.md)]

La identidad de servicio administrada proporciona a los servicios de Azure una identidad administrada en Azure Active Directory. Puede usar esta identidad para autenticar a cualquier servicio que admita la autenticación de Azure AD, sin necesidad de tener credenciales en el código. 

En este momento, no es posible crear, enumerar o eliminar una identidad administrada asignada por el usuario mediante Azure Portal.  Compruebe si hay actualizaciones.
