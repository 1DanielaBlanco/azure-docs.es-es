# Introducción al uso de HDInsight de Azure

[¿Qué es Azure AD?](active-directory-whatis.md)<br> [¿Cómo funciona?](active-directory-works.md)<br> [Pasos siguientes](active-directory-next-steps.md)<br> [Más información](active-directory-learn-map.md)


¿Ya está listo para comenzar? Azure AD está disponible para todos los usuarios de forma gratuita. Inscríbase ahora para obtener una suscripción de prueba gratuita de Azure y el directorio estará esperando por usted. El directorio es suyo independientemente de si compra algo o no. [Pruebe ahora Azure AD](http://azure.microsoft.com/trial/get-started-active-directory/)

## ¿Cómo debo decidir qué directorio usar para mis pruebas, el almacenamiento provisional y la producción?

Si tiene una suscripción existente para los servicios en la nube de Microsoft, como Office 365, ya dispondrá de un directorio en Azure AD y puede usar ese. Ese directorio ya autentica a los usuarios que inician sesión en Office 365. Solo necesitará suscribirse a Azure usando la misma cuenta de trabajo o colegio que usó para iniciar sesión en Office 365. A continuación, puede administrar el directorio en el portal de administración de Azure y agregar directorios para pruebas y demás según sea necesario. [Obtenga más información sobre cómo administrar el directorio de su suscripción de Office 365 en Azure](active-directory-how-subscriptions-associated-directory.md#manage-the-directory-for-your-office-365-subscription-in-azure).

Si aún no tiene una suscripción para un servicio de nube de Microsoft, se realizará el aprovisionamiento de un nuevo directorio para su trabajo o colegio al suscribirse a Azure y podrá usarlo como directorio principal.


## ¿Cómo puedo agregar mi propio nombre de dominio registrado?
¿Desea agregar un nombre de dominio personalizado o usar su propio nombre de dominio registrado? [Obtenga más información sobre la adición de un dominio personalizado](active-directory-add-domain.md).

## ¿Cómo puedo agregar mi marca de empresa?
Con Azure AD Premium, puede agregar el logotipo de su empresa y otras marcas a las páginas del panel de acceso y de inicio de sesión de Azure. [Incorporación de la marca de empresa a sus páginas de inicio de sesión y panel de acceso](active-directory-add-company-branding.md)

## ¿Cómo puedo integrar Azure AD con otro directorio?
Puede configurar la sincronización de directorios entre un directorio local y Azure AD, incluida la sincronización de contraseña y activar el inicio de sesión único para el acceso a las aplicaciones locales y en la nube. [Obtenga más información acerca de la integración de directorios](active-directory-aadconnect.md).

## ¿Cómo puedo crear y editar usuarios?
Cree una cuenta para cada usuario que vaya a tener acceso a un servicio de nube de Microsoft. De forma predeterminada, los usuarios no tienen permisos de administrador, pero puede asignárselos. [Obtenga más información sobre cómo crear usuarios de Azure AD](active-directory-create-users.md).

## Visualización de los informes de acceso y uso
Azure AD proporciona una serie de informes para los administradores. Hay más informes disponibles en las ediciones de pago de Azure AD y hay una API de generación de informes para crear sus propios informes personalizados. [Obtenga más información acerca de los informes de Azure AD](active-directory-view-access-usage-reports.md).
