# <a name="view-all-existing-groups-in-azure-active-directory-preview"></a>Visualización de todos los grupos existentes en la versión preliminar de Azure Active Directory
En este artículo se explica cómo ver todos los grupos en la versión preliminar de Azure Active Directory (Azure AD). [¿Qué es la versión preliminar?](active-directory-preview-explainer.md)  Una de las características de la administración de usuarios de Azure Active Directory (Azure AD) es la posibilidad de crear grupos de usuarios que pueden rellenarse con usuarios. Los grupos se usan para realizar tareas de administración, como asignar licencias o permisos a varios usuarios a la vez.

## <a name="how-do-i-see-all-the-groups"></a>¿Cómo puedo ver todos los grupos?
1. Inicie sesión en [Azure Portal](https://portal.azure.com) con una cuenta que tenga el rol de administrador global en el directorio.
2. Seleccione **Más servicios**, escriba **Usuarios y grupos** en el cuadro de texto y presione **Entrar**.
   
   ![Apertura de Administración de usuarios](./media/active-directory-groups-view-azure-portal/search-user-management.png)
3. En la hoja **Usuarios y grupos**, seleccione **Todos los grupos**.
   
   ![Apertura de la hoja Grupos](./media/active-directory-groups-view-azure-portal/view-groups-blade.png)
4. En la hoja **Usuarios y grupos - Todos los grupos** , puede agregar o quitar columnas de visualización, filtrar la lista para buscar un grupo o realizar cambios en los grupos de los que tiene permisos de modificación suficientes.
