# <a name="privileged-identity-management---resource-roles---configure"></a>Privileged Identity Management: configuración de roles de recursos

La configuración de los roles define los valores predeterminados aplicados a las asignaciones en el entorno PIM. Para definirlos para un recurso, seleccione la pestaña Configuración de rol desde el panel izquierdo o el botón de configuración del rol de la barra de acción de cualquier rol para ver las opciones actuales.


## <a name="select-a-role"></a>Seleccione un rol.
Seleccione un rol en el que le gustaría realizar un cambio. A continuación, seleccione Editar para realizar cambios en la configuración existente.
