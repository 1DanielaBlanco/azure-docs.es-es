# <a name="enabling-azure-active-directory-identity-protection"></a>Habilitación de Azure Active Directory Identity Protection
Azure Active Directory Identity Protection es una nueva funcionalidad que proporciona una vista consolidada de las actividades de inicio de sesión sospechosas y las posibles vulnerabilidades, además de notificaciones, recomendaciones de solución y directivas basadas en riesgos que lo ayudarán a proteger su negocio. 

El servicio detecta las actividades sospechosas de usuarios finales e identidades con privilegios (administrador) según determinadas señales, como ataques por fuerza bruta, pérdida de credenciales, inicios de sesión de ubicaciones desconocidas y dispositivos infectados. El objetivo es ofrecer una protección en tiempo real frente a estas actividades. Y lo que es más importante, se calcula la gravedad del riesgo para el usuario según estas actividades sospechosas y pueden configurarse directivas basadas en riesgos para proteger automáticamente las identidades de su organización. Para más información, consulte [Azure Active Directory Identity Protection](active-directory-identityprotection.md).

En este tema se muestra cómo habilitar Azure Active Directory Identity Protection.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Pasos para habilitar Azure Active Directory Identity Protection
1. [Inicie sesión](https://ms.portal.azure.com/) en Azure Portal como administrador. 
2. En Azure Portal, haga clic en **Marketplace**.
   
    ![Crear](./media/active-directory-identityprotection-enable/01.png "Create")
3. En la lista de aplicaciones, haga clic en **Seguridad e identidad**.
   
    ![Crear](./media/active-directory-identityprotection-enable/02.png "Create")
4. Haga clic en **Azure AD Identity Protection**.
   
    ![Crear](./media/active-directory-identityprotection-enable/03.png "Create")
5. En la hoja **Azure AD Identity Protection**, haga clic en **Crear**.
   
    ![Crear](./media/active-directory-identityprotection-enable/04.png "Create")
