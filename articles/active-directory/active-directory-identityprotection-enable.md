# <a name="enabling-azure-active-directory-identity-protection"></a>Habilitación de Azure Active Directory Identity Protection

Azure Active Directory Identity Protection es una función de Azure Active Directory (Azure AD). Con Azure AD Identity Protection, podrá hacer lo siguiente:

- Obtener una vista consolidada de los usuarios marcados y los eventos de riesgo detectados con algoritmos de aprendizaje automático.
- Establecer directivas de acceso condicional para proteger a los usuarios automáticamente.
- Mejorar la posición de seguridad actuando en función de las vulnerabilidades. 

Para obtener más información, consulte [Azure Active Directory Identity Protection](active-directory-identityprotection.md).

En este artículo se muestra cómo habilitar Azure Active Directory Identity Protection.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Pasos para habilitar Azure Active Directory Identity Protection
1. [Inicie sesión](https://ms.portal.azure.com/) en Azure Portal como administrador. 
2. En Azure Portal, haga clic en **Marketplace**.
   
    ![Crear](./media/active-directory-identityprotection-enable/01.png "Crear")
3. En la lista de aplicaciones, haga clic en **Seguridad e identidad**.
   
    ![Crear](./media/active-directory-identityprotection-enable/02.png "Crear")
4. Haga clic en **Azure AD Identity Protection**.
   
    ![Crear](./media/active-directory-identityprotection-enable/03.png "Crear")
5. En la hoja **Azure AD Identity Protection**, haga clic en **Crear**.
   
    ![Crear](./media/active-directory-identityprotection-enable/04.png "Crear")
