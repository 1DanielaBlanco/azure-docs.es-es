# <a name="default-quotas-for-azure-cosmos-db"></a>Cuotas predeterminadas para Azure Cosmos DB

Azure Cosmos DB es una base de datos de escala global en el que se pueden escalar el rendimiento y almacenamiento para gestionar todo lo que requiera la aplicación. Si tiene alguna pregunta sobre la escala que Azure Cosmos DB proporciona, envíe un correo electrónico a askdocdb@microsoft.com.
