## <a name="virtual-machine-extensions"></a>Extensiones de máquina virtual

Las extensiones que se distribuyen con el producto no se actualizarán durante el proceso normal de revisión y actualización; por ello, le recomendamos que compruebe con frecuencia las actualizaciones que pueda haber. Otras extensiones solo están disponibles a través de la administración de Marketplace.
