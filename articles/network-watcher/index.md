# <a name="azure-network-watcher-documentation"></a>Documentación de Azure Network Watcher

Aprenda a usar Network Watcher, un servicio que le permitirá supervisar y diagnosticar condiciones en un nivel de escenario de red. Tutoriales, referencias de API y otra documentación muestran cómo obtener información sobre la red de Azure con herramientas como la captura de paquetes, la topología y mucho más.
