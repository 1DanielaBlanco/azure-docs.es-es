# <a name="what-products-can-be-published-via-the-cloud-partner-portal"></a>Productos que se pueden publicar a través de Cloud Partner Portal

Cloud Partner Portal es el centro de publicación de la imagen de máquina virtual certificada de Azure en Microsoft Azure Marketplace o la aplicación Cortana Intelligence/de soluciones de Microsoft R para Microsoft AppSource.  

Las imágenes de máquina virtual son imágenes preconfiguradas con un sistema operativo completamente instalado y una o varias aplicaciones. Para publicar la imagen de máquina virtual en Azure Marketplace, debe cumplir los [requisitos técnicos previos](https://docs.microsoft.com/azure/marketplace-publishing/marketplace-publishing-vm-image-creation-prerequisites) para certificarla.
