# <a name="virtual-machine-scale-sets-documentation"></a>Documentación sobre conjuntos de escalado de máquinas virtuales

Aprenda a usar los conjuntos de escalado de máquinas virtuales para implementar y administrar un conjunto de máquinas virtuales Windows o Linux de escalado automático idénticas. La documentación muestra cómo usar conjuntos de escalado para admitir servicios orientados a proceso intensivo, macrodatos y soluciones en contenedores.
