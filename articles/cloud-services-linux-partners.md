Soluciones de socios para Linux en Azure
========================================

Póngase en contacto con los socios incluidos en esta página para recibir ayuda durante el desarrollo de las soluciones de la nube basadas en Linux en Azure. Además, suscríbase a la [vista previa gratuita](https://account.windowsazure.com/PreviewFeatures).

Bitnami
-------

[![Bitnami](./media/cloud-services-linux-partners/bitnami.png)](http://bitnami.org/azure/get-started)

<http://bitnami.org/azure/get-started>

BitNami proporciona imágenes completamente configuradas de aplicaciones web populares que se pueden implementar en Azure con un solo clic. Las imágenes de WebPack y DevPack de BitNami para Azure ofrecen entornos completamente configurados y listos para ejecutarse a los desarrolladores web y de aplicaciones de forma gratuita. WebPack de BitNami agrupa Wordpress, Drupal y Joomla! para ofrecer a los desarrolladores una selección de las mejores plataformas para desarrollar sitios web y blogs. DevPack de BitNami proporciona un entorno de desarrollo completo con tiempos de ejecución en lenguajes Ruby on Rails, PHP, Django y Java y todas sus dependencias, lo único que tienen que hacer los desarrolladores es compilar aplicaciones web en la nube.

Ubuntu Juju
-----------

[![Juju](./media/cloud-services-linux-partners/juju.png)](http://www.ubuntu.com/cloud/azure)

<http://www.ubuntu.com/cloud/azure>

Tras reconocer que la informática en nube era el futuro y después de analizar lo que podría significar la adopción de la nube para las organizaciones de TI y los desarrolladores, hemos creado Juju, un innovador conjunto de herramientas de orquestación de servicios que permite encapsular en un acceso el conocimiento conectado a las dependencias, las relaciones y la configuración de la plataforma de una aplicación.

Un acceso de Juju es una colección de instrucciones escritas previamente que implementa un servicio en la nube. Ya hay más de 50 disponibles, para servicios como Hadoop, PostgreSQL y Drupal, entre otros. Y como todos quieren enviar una, la selección aumenta constantemente.

Juju está estrechamente unido al aprovisionamiento, aunque es independiente de la capa de infraestructura subyacente. La plataforma base se puede instalar en una nube pública o privada, o bien en hardware, y está lista para que los servicios se implementen, se controlen y se escalen con Juju.
