# <a name="azure-sql-database-works-in-your-environment"></a>Base de datos SQL de Azure funciona en su entorno
## <a name="connect-with-what-you-already-know"></a>Conéctese con lo que ya conoce
Lo último que quiere es aprender algo nuevo solo para conectarse a una base de datos. La base de datos SQL de Azure hace que esto sea muy fácil gracias a varios lenguajes y plataformas con las que está acostumbrado a trabajar.

![funciona en su entorno](./media/sql-database-works-in-your-environment/sql-database-works-in-your-environment.png)

Al elegir una base de datos, no debería tener que moverse a una nueva plataforma o aprender un lenguaje nuevo. Base de datos SQL incluye ejemplos de código que le ayudarán a empezar a trabajar, donde quiera que esté.  

[Obtener más información sobre el desarrollo de Base de datos SQL](sql-database-develop-overview.md) 

## <a name="the-right-tools-for-the-job"></a>Las herramientas adecuadas para el trabajo
No tiene que aprender un nuevo conjunto de herramientas totalmente nuevo para empezar a trabajar con Base de datos SQL. Puede aprovechar herramientas de código abierto como [cheetah](https://github.com/wunderlist/cheetah), [sql-cli](https://www.npmjs.com/package/sql-cli) o [VS Code](https://code.visualstudio.com/). Además, Azure SQL Database funciona con herramientas de Microsoft como [Visual Studio](https://www.visualstudio.com/visual-studio-homepage-vs.aspx) y [SQL Server Management Studio](https://msdn.microsoft.com/library/ms174173.aspx).  También puede usar el Portal de administración de Azure, PowerShell y API de REST, que le ayudan a obtener más productividad.
