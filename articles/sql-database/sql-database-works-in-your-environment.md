# Base de datos SQL de Azure funciona en su entorno

## Conéctese con lo que ya conoce

Lo último que un desarrollador quiere es aprender algo nuevo solo para conectarse a una base de datos. La base de datos SQL de Azure hace que esto sea muy fácil gracias a varios lenguajes y plataformas con las que está acostumbrado a trabajar.

![funciona en su entorno](./media/sql-database-works-in-your-environment/sql-database-works-in-your-environment.png)

## JavaScript, Node.JS y Python, por nombrar unos cuantos

Al elegir una base de datos, no debería tener que aprender un lenguaje nuevo. Base de datos SQL tiene muchos ejemplos de código para ayudarle a empezar, para la plataforma en la que está trabajando. (es decir, Linux, Mac OS o Windows). Además, el Portal de administración de Azure, PowerShell y API de REST le ayudan a obtener más productividad al desarrollar.

[Lea la documentación](sql-database-develop-overview.md)

## Las herramientas adecuadas para el trabajo

Con Base de datos SQL no tiene que aprender a manejar una nueva herramienta cuando está desarrollando una aplicación. Puede aprovechar herramientas de código abierto como [cheetah](https://github.com/wunderlist/cheetah), [sql cli](https://www.npmjs.com/package/sql-cli) o [VS Code](https://code.visualstudio.com/). Además, Base de datos SQL de Azure funciona con herramientas de Microsoft como [Visual Studio](https://www.visualstudio.com/visual-studio-homepage-vs.aspx) y [SQL Server Management Studio](https://msdn.microsoft.com/library/ms174173.aspx).

## Introducción rápida

Con unos recursos garantizados, tutoriales sobre la creación de la primera base de datos SQL, consultas de solución de problemas y ejemplos de código de inicio rápido &mdash;obtener la respuesta correcta a su pregunta sobre la base de datos nunca es difícil.

[Lea la documentación](sql-database-technical-overview.md)

Vaya a la página principal para ver qué mas puede ofrecerle Base de datos SQL. [Compruébelo.](https://azure.microsoft.com/services/sql-database/)
