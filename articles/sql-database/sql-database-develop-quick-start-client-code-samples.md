# Ejemplos de código de desarrollo de cliente y de inicio rápido para la base de datos SQL


Este tema proporciona vínculos a ejemplos de código de inicio rápido que puede usar para conectarse a la base de datos SQL de Azure. Los ejemplos abarcan:


- Una amplia variedad de lenguajes de programación.
- Los sistemas operativos Windows y Linux como sistemas en los que puede ejecutarse el programa cliente.
- Ejemplos de código de inicio rápido breve, junto con ejemplos más largos que permiten controlar los errores transitorios mediante lógica de reintento automático.
- Ejemplos de código que permiten convertir conjuntos de resultados relacionales en un formato orientado a objetos.


> [AZURE.NOTE]Desde el 19 de mayo de 2015, se están preparando ejemplos de código en más lenguajes cuyos vínculos se agregarán al presente tema.
