# <a name="azure-sql-database-learns-amp-adapts"></a>Azure SQL Database aprende y se adapta
## <a name="how-azure-sql-database-learns-adapts-and-grows-with-your-application"></a>Cómo Base de datos SQL de Azure aprende, se adapta y crece con su aplicación
Base de datos SQL de Azure administra miles de millones de transacciones y millones de bases de datos al día. Y una de las características principales es que Base de datos SQL continuamente aprende y se adapta a su aplicación. De este modo, puede maximizar dinámicamente el rendimiento, la confiabilidad y la seguridad de los datos&mdash;con poco esfuerzo por su parte.

![aprende y adapta](./media/sql-database-learn-and-adapt/sql-database-learn-and-adapt.png)

## <a name="threat-detection-and-alerts"></a>Alertas y detección de amenazas
Gracias a la detección de amenazas y anomalías, Base de datos SQL dispone de análisis del comportamiento integrado, alertas en tiempo real, una directiva configurable de amenazas, un registro de auditoría y formas inteligentes de detectar y corregir los patrones poco habituales.

[Lea la documentación](sql-database-threat-detection-get-started.md)

## <a name="automatic-tuning"></a>Ajuste automático
Base de datos SQL está equipado para hacer que la aplicación se ejecute con el mejor rendimiento. Debido al aprendizaje continuo de los patrones de su aplicación, al ajuste automático adaptable del rendimiento y a la restricción automática no tiene que hacer nada. Y sabemos que eso es justo lo que quiere.

[Lea la documentación](http://go.microsoft.com/fwlink/?LinkID=787566)

## <a name="insights-when-you-need-them"></a>Información cuando la necesita
Base de datos SQL es capaz de realizar un seguimiento de cada consulta y su duración, frecuencia y utilización de recursos. En función de esta telemetría, los algoritmos automáticos ajustan de forma óptima las bases de datos a las consultas. Además, Base de datos SQL proporciona información para ayudar a minimizar los problemas de consultas de ajuste de tiempo y de rendimiento de la solución de problemas. Eso significa que dispone de un conocimiento directo sobre el consumo de recursos, las consultas de mayor rendimiento y la capacidad de profundizar para obtener más detalles.

[Lea la documentación](http://go.microsoft.com/fwlink/?LinkID=787567)

## <a name="no-administration-required"></a>Sin necesidad de administración
SQL Database ofrece la administración automática y la protección de datos que su aplicación necesita, por lo que puede volver a dedicarse a lo que mejor sabe hacer: desarrollar código. Conseguirá automáticamente copias de seguridad, conmutación por error en casos de recuperación ante desastres, mantenimiento de la infraestructura, seguridad y revisiones de software y actualizaciones de características. No solo ahorrará tiempo y dinero con Base de datos SQL funcionando en segundo plano, sino que además nunca más tendrá que preocuparse por los temibles tiempos de inactividad de la aplicación.

[Lea la documentación](http://go.microsoft.com/fwlink/?LinkID=787568)
