# <a name="manage-volumes-by-using-azure-netapp-files"></a>Administración de volúmenes mediante Azure NetApp Files
Puede administrar un volumen mediante la visualización, actualización o eliminación del volumen. 

Si desea eliminar un volumen, primero debe eliminar todos los objetos secundarios de un volumen. 

1.  Haga clic en la hoja **Volúmenes** de la hoja Manage Capacity Pools (Administración de grupos de capacidad). 
2.  Use la vista **Examinar volúmenes** para ver, actualizar o eliminar los volúmenes existentes.
