# Proveedores de almacenamiento, redes y procesos de Azure para aplicaciones de Windows en el modelo de implementación de Azure Resource Manager

La inclusión de funcionalidades de proceso, red y almacenamiento con el modelo de implementación de Azure Resource Manager simplificará esencialmente la implementación y la administración de aplicaciones complejas que se ejecutan en IaaS. Muchas aplicaciones requieren una combinación de recursos, como una red virtual, una cuenta de almacenamiento, una máquina virtual y una interfaz de red. El modelo de implementación de Azure Resource Manager ofrece la posibilidad de crear una plantilla JSON para implementar y administrar todos estos recursos como una sola aplicación.
