## <a name="working-with-images"></a>Trabajo con imágenes
Puede usar la Interfaz de la línea de comandos de Azure (CLI) para Mac, Linux y Windows para administrar las imágenes disponibles para su suscripción de Azure. También puede usar el Portal de Azure clásico para algunas tareas de imagen, pero la línea de comandos le ofrece más opciones.

Para ver ejemplos del uso de las herramientas, consulte [Comandos de la CLI de Azure comunes en Linux y Mac](virtual-machines-linux-cli-manage.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
