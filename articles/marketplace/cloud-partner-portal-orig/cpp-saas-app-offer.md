# <a name="azure-saas-application-offer"></a>Oferta de aplicación SaaS de Azure

<table> <tr> <td>En esta sección se explica cómo publicar una aplicación de software como servicio (SaaS) en Microsoft <a href="https://azuremarketplace.microsoft.com/marketplace/">Azure Marketplace</a> o <a href="https://appsource.microsoft.com">Microsoft AppSource</a>. Use aplicaciones SaaS cuando se implementa la solución en su propia suscripción de Azure y los clientes inician sesión mediante una interfaz que usted diseña y administra para probar la aplicación. Se utiliza <a href="https://azure.microsoft.com/services/active-directory">Azure Active Directory</a> (AAD) para aprovechar el entorno de prueba existente. </td> <td><img src="./media/cloud-partner-portal-publish-web-app/saasapp-icon1.png"  alt="Azure SaaS application icon" /></td> </tr> </table>


## <a name="publishing-benefits"></a>Ventajas de publicación

Algunas de las ventajas de anunciar las aplicaciones en Microsoft Marketplace incluyen:

- Llegue a 100 millones de usuarios de Azure Active Directory en Office 365 y Dynamics 365.
- Amplíe su equipo de ventas: llegue a usuarios de empresa de todo el mundo y obtenga un canal de ventas que interactúa con los usuarios finales, ayuda a generar clientes potenciales e inicia conversaciones con nuevos clientes de diferentes sectores.

- Obtenga información práctica: compartimos información detallada sobre cómo evoluciona su aplicación en AppSource, qué funciona bien y cómo mejorar más los procedimientos de ventas.


## <a name="publishing-process"></a>Proceso de publicación

Para crear y publicar sus aplicaciones en Microsoft Azure Marketplace o Microsoft AppSource, esta sección le guiará por los pasos siguientes:

1. Confirme que cumple los requisitos previos, incluidos los registros de asociados: [registro en el centro para desarrolladores](https://dev.windows.com/en-us/registration?accountProgram=Azure) *o* [registro en Microsoft Partner Network](https://partner.microsoft.com/).
2.  Defina los planes y la oferta de Marketplace.
3.  Proporcione precios y contenido de marketing.
4.  Pruebe y certifique su oferta en [Cloud Partner Portal](https://cloudpartner.azure.com).
5.  Empiece a trabajar con su oferta.
