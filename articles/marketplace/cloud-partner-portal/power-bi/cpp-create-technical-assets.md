# <a name="create-power-bi-app-technical-assets"></a>Creación de los recursos técnicos de la aplicación Power BI

El recurso técnico principal que necesitará para este tipo de oferta es una aplicación Power BI, que es una colección de un conjunto de datos principal, un informe o panel y servicios conectados y conjuntos de datos insertados opcionales. (Las aplicaciones Power BI son una evolución del tipo de recurso conocido anteriormente como [paquetes de contenido](https://docs.microsoft.com/power-bi/service-organizational-content-pack-introduction)).  Para obtener información completa sobre el desarrollo de estas aplicaciones, consulte [Aplicaciones Power BI](https://go.microsoft.com/fwlink/?linkid=2028636).


## <a name="obtaining-an-installation-url"></a>Obtención y dirección URL de instalación

La creación de una aplicación Power BI solo está disponible en el [servicio Power BI](https://powerbi.microsoft.com/) y requiere que inicie sesión con una [licencia de Power BI Pro](https://docs.microsoft.com/power-bi/service-admin-purchasing-power-bi-pro). Una vez que la aplicación de servicio se haya creado y probado en Power BI, guarde la dirección URL de instalación de la aplicación que genera Power BI.  Debe proporcionar esta dirección URL en la [pestaña Información técnica](./cpp-technical-info-tab.md) del formulario de envío de Cloud Partner Portal.
 

## <a name="next-steps"></a>Pasos siguientes

Después de crear y probar la aplicación Power BI y crear la oferta asociada, puede [publicar la oferta de Power BI](./cpp-publish-offer.md).
