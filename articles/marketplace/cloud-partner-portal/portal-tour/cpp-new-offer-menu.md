# <a name="new-offer-menu"></a>Menú Nueva oferta

El menú **Nueva oferta** permite a los asociados crear instancias de tipos de ofertas admitidos por [Cloud Partner Portal](https://cloudpartner.azure.com).  Al seleccionar el elemento de menú **Nueva oferta**, en la barra de menús de navegación del lado izquierdo, se muestra la siguiente lista de tipos de oferta.  Si selecciona uno de estos tipos, comienza el proceso de creación y publicación de la oferta.

|    **Elemento de menú Nueva oferta**     |     **Sección del documento correspondiente**                       |
|    -----------------------     |     -----------------------------                       |
| ![Elemento de menú Aplicaciones de Azure](./media/new-offer-menu1.png) |  [Oferta de aplicación de Azure](./../../cloud-partner-portal-orig/cpp-azure-app-offer.md) |
| ![Elemento de menú Servicios de consultoría](./media/new-offer-menu2.png) | [Oferta de servicios de consultoría](./../../cloud-partner-portal-orig/cloud-partner-portal-consulting-services-publishing-offer.md) |
| ![Elemento del menú Containers](./media/new-offer-menu3.png) | [Oferta de Containers](./../containers/cpp-containers-offer.md) |
| ![Elemento de menú Dynamics 365 Business Central](./media/new-offer-menu4.png) | [Oferta de Dynamics 365 Business Central](./../../cloud-partner-portal-orig/cpp-business-central-offer.md) |
| ![Elemento de menú Dynamics 365 for Customer Engagement](./media/new-offer-menu5.png) | [Oferta de Dynamics 365 for Customer Engagement](./../../cloud-partner-portal-orig/cpp-customer-engagement-offer.md) |
| ![Elemento de menú Dynamics 365 for Operations](./media/new-offer-menu6.png) | [Oferta de Dynamics 365 for Operations](./../../cloud-partner-portal-orig/cpp-dynamics-365-operations-offer.md) |
| ![Elemento de menú Módulos de IoT Edge](./media/new-offer-menu7.png) | [Oferta de Módulo de IoT Edge](./../iot-edge-module/cpp-offer-process-parts.md) |
| ![ Elemento de menú Aplicaciones SasS](./media/new-offer-menu8.png) | [Oferta de aplicación SaaS](./../../cloud-partner-portal-orig/cpp-saas-app-offer.md) |
| ![ Elemento de menú Máquinas virtuales](./media/new-offer-menu9.png) | [Oferta de máquina virtual](./../virtual-machine/cpp-virtual-machine-offer.md) |
|  |  |

> [!NOTE]
> Dado que este menú se actualiza periódicamente para reflejar los nuevos tipos de oferta y los cambios de nombre de las ofertas, los elementos que aparecen pueden no coincidir exactamente con los de la tabla anterior.


### <a name="next-steps"></a>Pasos siguientes

Si va a crear una nueva oferta, puede usar la tabla anterior para navegar hasta la sección de ofertas como guía para la creación y publicación de la oferta.  En caso contrario, el recorrido por el portal continúa con un examen de la [página Todas las ofertas](./cpp-all-offers-page.md).
