# Aplicaciones de ejemplo de Aprendizaje automático en Azure Marketplace

El Aprendizaje automático de Azure ofrece a los científicos de datos un nuevo canal para monetizar su saber y su creatividad igual que hacen las aplicaciones de smartphone para los desarrolladores. Al publicar sus servicios de aprendizaje automático en Azure Marketplace, permite que cualquier persona del mundo pague y se suscriba a sus API en la nube recientemente publicadas y las usen en sus aplicaciones.

Pero Azure Marketplace puede ofrecer mucho más que API de análisis básico. Por ejemplo, fuimos y creamos un conjunto completo de aplicaciones de Aprendizaje automático finalizadas, usando todas Aprendizaje automático de Azure, para resolver necesidades empresariales específicas. Estas aplicaciones de Aprendizaje automático no requieren un científico de datos ni un experto en Aprendizaje automático; la ciencia ya está integrada en nuestra solución. Los usuarios pueden traer sus propios datos y comenzar a usarlos. 

Estos incluyen por ejemplo:

* [Recomendaciones](http://datamarket.azure.com/dataset/amla/recommendations): un motor de recomendaciones de servicio automático muy potente que le permite agregar fácilmente recomendaciones de producto personalizadas a sus campañas de sitios web o de correo electrónico. 
  
* [Artículos que con frecuencia se compran juntos](https://datamarket.azure.com/dataset/amla/mba): una API que ayuda a los clientes a descubrir los artículos de su catálogo que con frecuencia se compran juntos. Utilice su historial de compras de clientes para agregar recomendaciones de "Artículos que con frecuencia se compran juntos" a su sitio web y para mejorar la conversión en su almacén digital.

* [Detección anómala](https://datamarket.azure.com/dataset/aml_labs/anomalydetection): una API que puede colocar los eventos anómalos en datos de series temporales (por ejemplo, telemetría de servidores, datos basados en sensores).

Estas API permiten consumir potentes servicios basados en el aprendizaje automático creados en el Aprendizaje automático de Azure, sin necesidad de ser un científico de los datos o de pasar mucho tiempo desarrollándolos, basta con traer sus propios datos y ya puede comenzar a usarlos.
