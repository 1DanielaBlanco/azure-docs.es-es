# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Cómo suscribirse a Microsoft Translator Speech API

Siga estos pasos para suscribirse a Microsoft Translator Speech API.

## <a name="1-sign-into-azure"></a>1: Iniciar sesión en Azure   
- ¿No tiene una cuenta? [Suscríbase a una cuenta de Microsoft Azure](http://azure.com/)
- ¿Ya tiene una cuenta? [Sign in](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2. Suscribirse a Microsoft Translator     
- Después de iniciar sesión en Azure, vaya a la sección [Cognitive Services](https://portal.azure.com/#create/Microsoft.CognitiveServices)
- En "Tipo de API", seleccione Translator Speech API
- En la sección de plan de tarifa, seleccione el plan de tarifa que mejor se adapte a sus necesidades
    - Cada suscripción tiene un plan gratuito. El plan gratuito tiene las mismas características y funcionalidades que los planes de pago, y no tiene una fecha de expiración.
    - Solo puede tener una suscripción gratuita en su cuenta
- Rellene el resto del formulario y seleccione el botón Crear
- La suscripción está lista para su uso inmediato

## <a name="3-authentication-key"></a>3: Clave de autenticación    
- Recupere la clave de autenticación; para ello, vaya al menú Todos los recursos > haga clic en su suscripción
- El valor de clave se usa para la autenticación. Mantenga el valor de clave confidencial. Necesita el valor de clave para desarrollar su aplicación

- [Foro de soporte técnico de Microsoft Translator](http://www.aka.ms/TranslatorForum)
