# <a name="get-the-cognitive-services-speech-devices-sdk"></a>Obtener el SDK de dispositivos de voz de Cognitive Services

## <a name="requesting-access"></a>Solicitud del acceso

El SDK de dispositivos de voz está en versión preliminar restringida y requiere que esté inscrito en el programa. En la actualidad, Microsoft prefiere a las grandes empresas como candidatos para el acceso a este producto.

Para obtener acceso al SDK de dispositivos de voz, siga estos pasos:

1. Vaya al [formulario de registro](https://aka.ms/sdsdk-signup) del SDK de dispositivos de voz de Microsoft.
1. Lea el [contrato de licencia](speech-devices-sdk-license.md).
1. Si acepta los términos de este contrato de licencia, seleccione "Acepto".
1. Responda a las preguntas del formulario.
1. Envíe el formulario. 
1. Si su dirección de correo electrónico todavía no forma parte de Azure Active Directory, recibirá un correo electrónico de invitación como el siguiente tras la aprobación. Si la dirección de correo electrónico ya está en Azure Active Directory, recibirá un mensaje de correo electrónico del equipo de Microsoft Speech tras la aprobación y podrá ir directamente a la [descarga del SDK de dispositivos de voz](#download-the-speech-devices-sdk).

## <a name="approval-e-mail"></a>Correo electrónico de aprobación

```
From: Microsoft Speech Team from Microsoft (via Microsoft) <invites@microsoft.com> 
Subject: You're invited to the Microsoft organization 
```

![Mensaje de correo electrónico](media/speech-devices-sdk/get-sdk-1.png)

## <a name="accept-access"></a>Aceptación de acceso
Siga los pasos siguientes para unirse a Azure Active Directory con la dirección de correo electrónico que proporcionó durante el registro. Este proceso concede acceso al [sitio de descarga](https://shares.datatransfer.microsoft.com/) del SDK de dispositivos de voz.

1. Haga clic en **Comenzar** en el mensaje de correo electrónico que recibió. Si la organización ya es un cliente de Office 365, se le pedirá que inicie sesión y puede ir directamente al paso 8.

2. Haga clic en **Siguiente** en la ventana del explorador que se inicia.

    ![Ventana de autenticación](media/speech-devices-sdk/get-sdk-2.png)

3. Cree una cuenta de Microsoft si aún no tiene una. Escriba la misma dirección de correo electrónico en la que ha recibido el correo electrónico de invitación en el paso 6 anterior.

    ![Creación de una cuenta de Microsoft](media/speech-devices-sdk/get-sdk-3.png)

4. Haga clic en **Siguiente** para crear una contraseña.

5. Cuando se le solicite que verifique el correo electrónico, vuelva a la Bandeja de entrada de correo electrónico para obtener el código de comprobación que se le envió.
 
7. Pegue o escriba el código de seguridad del mensaje de correo electrónico en el cuadro de diálogo. En este ejemplo, es "8406". A continuación, haga clic en **Siguiente**.

    ![Comprobar el correo electrónico](media/speech-devices-sdk/get-sdk-6.png)
 
8. Cuando aparezca la aplicación del panel de acceso en la ventana del explorador, habrá confirmado que la dirección de correo electrónico (del paso 6) ahora forma parte de Azure Active Directory. Ahora tiene acceso al sitio de descarga del SDK de dispositivos de voz.

## <a name="download-the-speech-devices-sdk"></a>Descargar el SDK de dispositivos de voz

Vaya al [sitio de descarga del SDK de dispositivos de voz](https://shares.datatransfer.microsoft.com/) e inicie sesión con la cuenta de Microsoft que creó anteriormente. Ahora puede descargar el SDK de dispositivos de voz, el código de ejemplo asociado y el material de referencia siguiendo estos pasos.

![Sitio de descarga del SDK](media/speech-devices-sdk/get-sdk-7.png)

1. Descargue e instale la herramienta Aspera Connect cuando el explorador le pida que lo haga.

    ![Descargar Aspera Connect](media/speech-devices-sdk/get-sdk-8.png)
 
1. Haga clic en **Sí** para cambiar a Aspera Connect.

    ![Cambiar a Aspera Connect](media/speech-devices-sdk/get-sdk-9.png)
 
1. Haga clic en **Permitir** para confirmar la descarga de los archivos con Aspera Connect.

    ![Descarga con Aspera Connect](media/speech-devices-sdk/get-sdk-10.png)
 
1. Cierre la ventana de transferencias de Aspera Connect después de que se hayan descargado los archivos.

    ![Ventana de transferencias de Aspera Connect](media/speech-devices-sdk/get-sdk-11.png)
 
De forma predeterminada, los archivos se descargan en la carpeta **Descargas**. Ya puede cerrar la sesión en este sitio.
