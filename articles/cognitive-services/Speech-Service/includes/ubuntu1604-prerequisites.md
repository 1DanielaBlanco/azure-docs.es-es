En Ubuntu 16.04, asegúrese de que tiene el compilador y las bibliotecas necesarios. Para ello, ejecute los siguientes comandos:

```sh
sudo apt-get update
sudo apt-get install build-essential libssl1.0.0 libcurl3 libasound2 wget
```
