> [!NOTE]
> Primero es necesario que obtenga una clave de suscripción.
> Puede obtener claves de suscripción de prueba gratuitas en la página de [suscripción a Cognitive Services](https://azure.microsoft.com/try/cognitive-services/). Después de seleccionar Speech API, elija **Get API Key** (Obtener clave de API) para obtener dos claves (principal y secundaria). Puede usar cualquiera de las claves. Las dos claves están asociadas a la misma cuota.
>
> Las claves de suscripción de pago están disponibles a través de su panel de Azure.
