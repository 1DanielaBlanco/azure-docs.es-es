# <a name="glossary"></a>Glosario

## <a name="a"></a>Una 

### <a name="acoustic-model"></a>Modelo acústico

El modelo acústico es un clasificador que etiqueta fragmentos cortos de audio en uno de una serie de fonemas, o unidades de sonido, en un idioma determinado. Por ejemplo, la palabra "voz" se compone de tres fonemas: "b o z". Estas clasificaciones se llevan a cabo a razón de 100 veces por segundo.

## <a name="b"></a>b

## <a name="c"></a>C

### <a name="conversational-model"></a>Modelo conversacional

Un modelo adecuado para el reconocimiento de voz en un estilo conversacional. El modelo acústico conversacional de Microsoft está adaptado para la voz dirigida normalmente a otra persona.

## <a name="d"></a>D

### <a name="deployment"></a>Implementación

El proceso a través del cual el modelo personalizado adaptado se convierte en un servicio y expone un identificador URI.

## <a name="i"></a>I

### <a name="inverse-text-normalization"></a>Normalización inversa de texto

El proceso de conversión de texto sin formato "sin procesar" nuevamente a texto con formato, es decir, con mayúsculas y signos de puntuación, se denomina normalización inversa de texto (ITN).

## <a name="j"></a>J

## <a name="k"></a>K

## <a name="l"></a>L

### <a name="language-model"></a>Modelo de lenguaje

El modelo de lenguaje es una distribución de probabilidad sobre secuencias de palabras. Ayuda al sistema a decidir entre secuencias de palabras que suenan de forma parecida, en función de la probabilidad de las propias secuencias de palabras.

## <a name="m"></a>M

## <a name="n"></a>N

### <a name="normalization"></a>Normalización

Normalización (texto): transformación del texto resultante (es decir, transcripción) en un formato estándar y no ambiguo, legible por el sistema.

## <a name="s"></a>S

### <a name="search-and-dictate-model"></a>Modelo de búsqueda y dictado

Modelo acústico adecuado para el procesamiento de comandos. El modelo acústico Microsoft Search and Dictation resulta adecuado para la voz dirigida a una aplicación o dispositivo, como comandos.

### <a name="subscription-key"></a>Subscription key

La clave de suscripción es una cadena que se debe especificar como parámetro de cadena de consulta para invocar cualquier modelo de Custom Speech Service. Se obtiene una clave de suscripción de [Azure Portal](https://portal.azure.com/#create/Microsoft.CognitiveServices/apitype/CustomSpeech) y, una vez obtenida, se puede encontrar en "My Subscriptions" (Mis suscripciones) en el portal de Custom Speech Service.

## <a name="t"></a>T

### <a name="transcription"></a>Transcripción

Transcripción: el fragmento de texto que es el resultado del proceso de un fragmento del archivo de audio .wav
