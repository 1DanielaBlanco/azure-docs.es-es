# <a name="bing-search-sdk"></a>SDK de Bing Search
Los ejemplos de Bing Image Search API incluyen los escenarios siguientes:
1.  Consulta de una imagen sencilla
2.  Búsqueda restringida a tipos de imágenes y relaciones de aspecto específicos, que muestra funcionalidades de filtrado de imágenes.
3.  Búsqueda de imágenes populares en varias redes sociales en este momento.
4.  Consulta de la información de una imagen, como el origen de la imagen, texto representativo de la consulta, etc.

Los SDK de Bing Search hacen que se pueda acceder de inmediato a la funcionalidad de búsqueda web en los siguientes lenguajes de programación:
* Introducción a [ejemplos de .NET](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [Paquete NuGet](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.ImageSearch/1.2.0)
    * Consulte también [bibliotecas de .NET](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingImageSearch) para las definiciones y dependencias.
* Introducción a [ejemplos de Node.js](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * Consulte también [bibliotecas de Node.js](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) para las definiciones y dependencias.
* Introducción a [ejemplos de Java](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * Consulte también [bibliotecas de Java](https://github.com/Azure/azure-sdk-for-java/tree/master/azure-cognitiveservices/search/bingimagesearch) para las definiciones y dependencias.
* Introducción a [ejemplos de Python](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * Consulte también [bibliotecas de Python](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) para las definiciones y dependencias.

Entre los ejemplos de SDK para cada lenguaje se incluye un archivo Léame con detalles sobre los requisitos previos y la instalación y ejecución de los ejemplos.
