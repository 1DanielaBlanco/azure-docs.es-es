# <a name="entity-search-endpoints"></a>Puntos de conexión de Entity Search
**Entity Search API** incluye un punto de conexión.

## <a name="endpoint"></a>Punto de conexión
Para solicitar los resultados de Entity Search, envíe una solicitud al punto de conexión siguiente. Use los encabezados y parámetros de dirección URL para definir especificaciones adicionales.

Punto de conexión `GET`: 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

Se necesitan los parámetros de dirección URL siguientes:
- mkt. Mercado de dónde proceden los resultados. 
- q. Consulta de Entity Search.
