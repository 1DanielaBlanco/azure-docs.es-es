# <a name="microsoft-speech-client-samples"></a>Ejemplos del cliente de Voz de Microsoft

El servicio Voz de Microsoft proporciona ejemplos completos que muestran cómo usar Microsoft Speech Recognition API en distintos casos de uso; por ejemplo, reconocimiento de comandos, reconocimiento continuo y la detección de intenciones. Todos los ejemplos están disponibles en GitHub y se pueden descargar con los vínculos siguientes: el archivo README.md que se encuentra en todos los repositorios y la página de [bibliotecas cliente](GetStarted/GetStartedClientLibraries.md) proporcionan información detallada sobre cómo compilar y ejecutar los ejemplos.

- [Ejemplos de API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Ejemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Ejemplos de clientes de escritorio de C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Ejemplos de servicios de C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Ejemplos de Java en Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [Ejemplos de ObjectiveC en iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos los SDK de Microsoft Cognitive Services y los ejemplos tienen una licencia de MIT. Para más información, consulte [LICENCIA](https://github.com/Microsoft/Cognitive-Speech-STT-JavaScript/blob/master/LICENSE.md).
