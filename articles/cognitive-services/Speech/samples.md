# <a name="bing-speech-client-samples"></a>Ejemplos de cliente de Bing Speech

[!INCLUDE [Deprecation note](../../../includes/cognitive-services-bing-speech-api-deprecation-note.md)]
 
Bing Speech proporciona ejemplos de un extremo a otro que muestran distintos casos de uso; por ejemplo, reconocimiento de comandos, reconocimiento continuo y detección de intenciones. Todos los ejemplos están disponibles en GitHub y se pueden descargar mediante los vínculos siguientes: En el archivo README.md de cada repositorio, así como en la página de [bibliotecas cliente](GetStarted/GetStartedClientLibraries.md) se proporcionan detalles sobre cómo compilar y ejecutar los ejemplos.

- [Ejemplos de API REST](https://github.com/Azure-Samples/SpeechToText-REST)

- [Ejemplos de JavaScript](https://github.com/Azure-Samples/SpeechToText-WebSockets-Javascript)

- [Ejemplos de clientes de escritorio de C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-Windows)

- [Ejemplos de servicios de C#](https://github.com/Azure-Samples/Cognitive-Speech-STT-ServiceLibrary)

- [Ejemplos de Java en Android](https://github.com/Azure-Samples/Cognitive-Speech-STT-Android)

- [Ejemplos de ObjectiveC en iOS](https://github.com/Azure-Samples/Cognitive-Speech-STT-iOS)

Todos los SDK de Microsoft Cognitive Services y los ejemplos tienen una licencia de MIT. Para más información, consulte [LICENCIA](https://github.com/Azure-Samples/SpeechToText-REST/blob/master/LICENSE).
