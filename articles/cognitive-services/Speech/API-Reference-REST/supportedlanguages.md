# <a name="language-and-region-support-for-speech-api"></a>Idiomas y regiones admitidos para Speech API

## <a name="interactive-and-dictation-mode"></a>Modo interactivo y dictado

Microsoft Speech Recognition API admite los siguientes idiomas en los modos `interactive` y `dictation`.

|Código | Idioma | Código | Idioma |
|-----|-----|-----|-----|
| ar-EG | Árabe (Egipto), estándar moderno | hi-IN | Hindi (India) |
| ca-ES | Catalán (España) | it-IT | Italiano (Italia)  |
| da-DK | Danés (Dinamarca) | ja-JP | Japonés (Japón) |
| de-DE | Alemán (Alemania) |ko-KR | Coreano (Corea) |
| en-AU | Inglés (Australia) |nb-NO | Noruego, Bokmål (Noruego)  |
| en-CA | Inglés (Canadá) | nl-NL | Neerlandés (Países Bajos)   |
| en-GB | Inglés (Reino Unido) |pl-PL | Polaco (Polonia) |
| en-IN | Inglés (India) | pt-BR | Portugués (Brasil)  |
| en-NZ | Inglés (Nueva Zelanda) |pt-PT | Portugués (Portugal)  |
| es-ES | Inglés (Estados Unidos) | ru-RU | Ruso (Rusia) |
| es-ES | Español (España) | sv-SE | Sueco (Suecia) |
| es-MX | Español (México) |zh-CN | Chino (mandarín, simplificado)  |
| fi-FI | Finés (Finlandia) |zh-HK | Chino (Hong Kong RAE) |
| fr-CA | Francés (Canadá) | zh-TW | Chino (mandarín, Taiwán)|
| fr-FR | Francés (Francia) | gu-IN | Gujarati (India) |
| ta-IN |Tamil (India) | te-IN | Telugu (India) |

| Código | Idioma | Código | Idioma |
|-----|-----|-----|-----|
| ar-EG | Árabe (Egipto), estándar moderno | It-IT | Italiano (Italia) |
| de-DE | Alemán (Alemania) | ja-JP | Japonés (Japón) |
| es-ES | Inglés (Estados Unidos) | pt-BR | Portugués (Brasil) |
| es-ES | Español (España) | ru-RU | Ruso (Rusia) |
| fr-FR | Francés (Francia) | zh-CN | Chino (mandarín, simplificado) |
