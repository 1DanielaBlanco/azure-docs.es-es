Antes de crear el ejemplo, debe suscribirse a [Anomaly Finder API](https://labs.cognitive.microsoft.com/en-us/project-anomaly-finder), que forma parte de Microsoft Cognitive Services. Haga clic en el botón "Suscribirse", luego inicie sesión con su cuenta de Microsoft (Hotmail, etc.) para suscribirse a la API y después encontrará las claves de la suscripción gratuita [aquí](https://labs.cognitive.microsoft.com/en-us/subscriptions?productId=/products/AnomalyDetection.Preview&source=labs). Se pueden utilizar la clave 1 y la clave 2.
