# <a name="what-is-the-linguistic-analysis-api-preview"></a>¿Qué es Linguistic Analysis API? (versión preliminar)

> [!IMPORTANT]
> El 9 de agosto de 2018 se retiró la versión preliminar de Linguistic Analysis. Para el procesamiento y el análisis de texto, se recomienda usar los [módulos de análisis de texto de Azure Machine Learning](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics).

Linguistic Analysis API proporciona acceso a las herramientas de procesamiento de lenguaje natural (NLP) que identifican la estructura del texto. La versión actual ofrece tres tipos de análisis:

- [Separación y tokenización de oraciones](Sentences-and-Tokens.md)
- [Etiquetado de categorías gramaticales](POS-tagging.md)
- [Análisis de circunscripción](Constituency-Parsing.md)

## <a name="getting-started"></a>Introducción

Cuando se haya familiarizado con los tipos principales de análisis mencionados anteriormente, puede buscar el punto de conexión para [obtener la lista de analizadores disponibles](AnalyzersMethod.md) y cómo se [nombran](Analyzer-Names.md) y, luego, el punto de conexión para [analizar la nueva entrada](AnalyzeMethod.md).
