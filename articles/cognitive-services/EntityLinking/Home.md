# <a name="entity-linking-intelligence-service-api"></a>Entity Linking Intelligence Service API

Esto es Microsoft Entity Linking Intelligence Service, un servicio web creado para ayudar a los desarrolladores en las tareas relacionadas con la vinculación de entidades.

## <a name="entity-linking"></a>Entity Linking

A veces, en diferentes contextos, se puede usar una palabra como una entidad con nombre, un verbo u otra palabra de una oración determinada. Por ejemplo, en el caso de que "times" sea una entidad con nombre, puede hacer referencia a dos entidades que pueden distinguirse por separado, como "The New York Times" o "Times Square". En un párrafo específico de un documento, Entity Linking Intelligence Service reconocerá e identificará cada entidad por separado en función del contexto.  

En la siguiente ilustración se muestra un ejemplo de vinculación de entidad. En concreto, cuando se usa Wikipedia, Entity Linking Intelligence Service detecta todas las entidades que se mencionan en el texto de entrada y las vincula a los puntos de referencia pertinentes según el identificador de página.
