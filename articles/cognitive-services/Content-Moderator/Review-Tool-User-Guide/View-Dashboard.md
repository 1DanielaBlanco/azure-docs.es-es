# <a name="view-dashboard"></a>Visualización del panel

Haga clic en la pestaña **Panel** a fin de ver las métricas clave para la revisión de todo el contenido realizada en la herramienta. Puede ver las revisiones en curso, completadas y totales para el texto y las imágenes. Puede ver las métricas para el equipo predeterminado y para los equipos secundarios.
