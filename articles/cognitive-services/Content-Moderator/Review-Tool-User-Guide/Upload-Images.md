# <a name="upload-images"></a>Carga de imágenes #

Para explorar la interfaz de usuario, cargue hasta cinco imágenes a la vez, sin que cada una de ellas supere los 2 MB en el tamaño de archivo. Asegúrese de comprobar el tamaño de los archivos antes de cargarlos.

![Carga de imágenes](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Envío para moderación ##
Si finalmente carga más de cinco imágenes a la vez, use el icono de la papelera para quitar las imágenes no deseadas. A continuación, envíe las imágenes para moderación automatizada.

![Envío de imágenes](images/1-try-image-2.png)
