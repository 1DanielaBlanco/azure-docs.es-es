    <h1>Documentación de la API de Text Analytics</h1>
    <div class="intro" style="min-width: 200px">
        <p>La API de Text Analytics es un servicio basado en la nube que proporciona un procesamiento avanzado de lenguaje natural sobre texto sin formato, e incluye tres funciones principales: análisis de sentimiento, extracción de frases clave y detección de idioma. Aprenda a analizar el contenido de varias maneras con nuestros inicios rápidos, tutoriales y ejemplos.</p>
    </div>
