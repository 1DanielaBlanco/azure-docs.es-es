# Métodos abreviados de teclado del Portal de Azure 

## Abrir centros

| Método abreviado | Acción |
|--------|----------|
| N | Abrir centro de notificaciones |
| S o / | Abrir centro de exploración (servicios) | 
| A | Seleccionar una sesión activa | 
| C | Crear | 

## Navegación

| Método abreviado | Acción |
|--------|----------|
| F | Moverse a la primera hoja abierta | 
| H | Cambiar entre la página principal del Portal de Azure y la hoja actual | 
| J | Moverse a la hoja abierta anterior | 
| K | Moverse a la hoja abierta siguiente | 
| L | Moverse a la última hoja abierta | 
| - | Minimizar la hoja o restaurar desde maximizada | 
| + | Maximizar la hoja o restaurar desde minimizada | 
| K | Moverse a la hoja abierta siguiente | 


## Controles
| Método abreviado | Acción |
|--------|----------|
| Esc | Cerrar un menú de centro, cuadro de diálogo, menú de cuenta o cualquier menú abierto | 
| # | Cerrar todas las hojas abiertas | 
| Inicio | Desplazarse a la parte superior de la hoja actual | 
| Fin | Desplazarse a la parte inferior de la hoja actual | 
| CTRL+M | Alternar entre la tabulación entre controles y el uso de tabulaciones dentro de un control de edición |
