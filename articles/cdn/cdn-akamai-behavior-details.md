# Detalles del comportamiento de la red CDN de Azure de Akamai

En este artículo se describen detalles de comportamiento permitido propios de la **red CDN de Azure de Akamai**.

> [AZURE.IMPORTANT] La información de este artículo no es aplicable a la red CDN de Azure de puntos de conexión de Verizon.

## Puertos de origen permitidos

La **red CDN de Azure de Akamai** permite el acceso a un subconjunto de puertos TCP en el servidor de origen. A continuación se muestra la lista de intervalos de puertos permitidos.
