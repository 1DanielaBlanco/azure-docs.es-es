# <a name="commands"></a>Comandos:
|||
|---|---|
|az mysql server|Administración de servidores. Los servidores contienen bases de datos, almacenamientos de datos y grupos elásticos.|
|az mysql server create|Crea un servidor MySQL.|
|az mysql server delete|Elimina un servidor MySQL.|
|az mysql server list|Obtención de una lista de todos los servidores MySQL en un grupo de recursos.|
|az mysql server show|Presentación de detalles de un servidor MySQL.|
|az mysql server update|Actualiza un servidor.|
|az mysql server restore|Restauración de un servidor a un momento anterior en el tiempo.|
|az mysql server firewall-rule|Administración de las reglas de firewall de un servidor.|
|az mysql server firewall-rule create|Crea una regla de firewall.|
|az mysql server firewall-rule delete|Elimina una regla de firewall.|
|az mysql server firewall-rule list|Enumera las reglas de firewall.|
|az mysql server firewall-rule show|Muestra los detalles de una regla de firewall.|
|az mysql server firewall-rule update|Actualiza una regla de firewall.|
|az mysql db|Administración de bases de datos.|
|az mysql db list|Enumeración de todas las bases de datos MySQL en un servidor.|
|az mysql server-logs list|Enumeración de los archivos de registro de un servidor.|
|az mysql server-logs download|Descarga de registros del servidor en el sistema local.|
|az mysql server configuration list|Enumeración de los parámetros de configuración de servidor.|
|az mysql server configuration show|Presentación de detalles de los parámetros de configuración de servidor.|
|az mysql server configuration update|Modificación del valor de los parámetros de configuración de servidor.|
|||
