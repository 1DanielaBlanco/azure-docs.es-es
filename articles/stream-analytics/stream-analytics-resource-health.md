# <a name="stream-analytics-resource-health-page-in-the-azure-portal"></a>Página Estado de los recursos de Stream Analytics en Azure Portal

La página **Estado de los recursos** de Azure Portal muestra el estado de los trabajos de Azure Stream Analytics. También muestra las instrucciones que pueden ayudarle a solucionar problemas. 

La página **Estado de los recursos** está diseñada para ayudarle a determinar más rápidamente si es la raíz de un problema está en la aplicación, o si lo ha producido un evento en Azure. Puede ayudarle a saber cuándo se debe abrir un caso de soporte técnico para recibir asistencia.

## <a name="troubleshooting-workflow"></a>Flujo de trabajo de solución de problemas

Para ver el estado de los recursos para los trabajos de Stream Analytics, seleccione la página **Estado de los recursos**. 

![Flujo de solución de problemas de Stream Analytics: acceso](media/stream-analytics-resource-health/stream-analytics-access-troubleshooting.png)

El diagrama de flujo siguiente puede ayudarle a tomar decisiones sobre qué se debe buscar y cuándo se debe abrir un caso de soporte técnico.

![Flujo de solución de problemas de Stream Analytics: asignación](media/stream-analytics-resource-health/stream-analytics-troubleshooting-map.png)
