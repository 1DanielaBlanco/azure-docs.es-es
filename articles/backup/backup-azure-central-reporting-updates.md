# <a name="updating-azure-backup-central-reporting-content-pack"></a>Actualización del paquete de contenido de informes central de Azure Backup 

El [Paquete de contenido de Azure Backup](https://docs.microsoft.com/azure/backup/backup-azure-configure-reports#view-reports-in-power-bi) se puede usar para ver informes sobre copias de seguridad de forma centralizada. El paquete de contenido se actualiza con regularidad para agregar más características y para corregir errores. Este artículo le guía por los pasos para actualizar el paquete de contenido, para retrasar la actualización y las actualizaciones que se realizan con el tiempo.

## <a name="how-to-get-updates-to-the-content-pack"></a>Cómo obtener actualizaciones para el paquete de contenido

### <a name="to-get-the-updated-content-pack"></a>Para obtener el paquete de contenido actualizado
La copia del paquete de contenido se actualizará automáticamente si no ha realizado algún cambio en él. En caso de que se haya cambiado el paquete de contenido, obtendrá una notificación en Power BI y una notificación por correo electrónico acerca de la misma. Puede obtener el paquete de contenido actualizado según su preferencia. 

### <a name="to-delay-the-update"></a>Para retrasar la actualización
El procedimiento recomendado consiste en importar el paquete de contenido en un [área de trabajo personalizada](https://youtu.be/26zyOtyHPJM?t=1m57s). Ahora tiene la capacidad de editar los informes.
Como se mencionó anteriormente, si se cambia el paquete de contenido, verá una notificación en Power BI. Puede elegir obtener el paquete de contenido más tarde. 

## <a name="coming-soon"></a>Próximamente
   
El paquete de contenido de Azure Backup se está actualizando para admitir más cargas de trabajo, como SQL en máquinas virtuales de IaaS y SC DPM, además de la compatibilidad actual con MAB y la copia de seguridad de la máquina virtual de Azure. Esto significa que pronto podrá ver y analizar todos los datos de copias de seguridad en un lugar central. [También se pueden personalizar los informes](https://youtu.be/26zyOtyHPJM) para satisfacer las necesidades de su organización.
