Administración de datos de Azure: Toma de la mejor decisión
===========================================================

**Resumen:** Azure ofrece varias formas de almacenar y administrar los datos. Esta diversidad permite a los usuarios de la plataforma solucionar una gran variedad de problemas distintos. Sin embargo, la diversidad implica tomar decisiones: ¿qué opción de administración de datos es la adecuada para una situación determinada?

Esta breve introducción se ha creado para ayudarle a responder esa pregunta. Obtenga un resumen rápido de las tecnologías de administración de datos en Azure: Base de datos SQL, SQL Server en una VM, Almacenamiento de blobs y Almacenamiento de tablas. A continuación, lea sobre los distintos escenarios y conozca cuál de estas es la tecnología más apropiada en cada caso.
