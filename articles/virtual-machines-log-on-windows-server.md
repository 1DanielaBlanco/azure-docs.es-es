#Inicio de sesión en una máquina virtual con Windows Server#

Para iniciar sesión en una máquina virtual con Windows Server, use el botón Conectar del Portal de administración para iniciar una conexión de escritorio remoto. (Para máquinas virtuales Linux, consulte [Inicio de sesión en una máquina virtual con Linux](virtual-machines-linux-how-to-log-on.md).)
