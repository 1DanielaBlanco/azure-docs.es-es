# <a name="azure-cosmos-db-documentation-experiment"></a>Experimento de documentación sobre Azure Cosmos DB

Azure Cosmos DB es una base de datos de varios modelos distribuidos globalmente. Dado que puede realizar numerosas acciones con Azure Cosmos DB (desde crear un gráfico hasta crear una base de datos de documentos o migrar datos de MongoDB, Cassandra o Azure Table Storage), organizar la documentación puede suponer un reto. Por eso, nos gustaría que nos ayudara.

Estamos probando la organización de la tabla de contenidos y la [página de aterrizaje](https://docs.microsoft.com/en-us/azure/cosmos-db/) de Azure Cosmos DB, tal y como se muestra en las siguientes imágenes, y nos gustaría que nos enviara sus comentarios sobre los cambios propuestos.

## <a name="to-load-both-views"></a>Para cargar ambas vistas

1. Para abrir ambas vistas en paralelo, vaya a la [página de aterrizaje](https://docs.microsoft.com/en-us/azure/cosmos-db/) desde su navegador web favorito. 
2. Utilice las imágenes anteriores para determinar qué versión ve, la original o la experimental. 
3. Abra una ventana privada o de incógnito del navegador y cargue la [página de aterrizaje](https://docs.microsoft.com/en-us/azure/cosmos-db/) en la nueva ventana del navegador y vuelva a utilizar las imágenes para determinar la versión que ve. 

   Si no tiene las dos vistas presentes, cierre y vuelva a abrir una ventana privada o de incógnito y vuelva a cargar la [página de aterrizaje](https://docs.microsoft.com/en-us/azure/cosmos-db/). Puede que deba intentarlo varias veces antes de cargar ambas versiones en ventanas independientes. Actualizar la ventana de incógnito no hará que se cargue la otra versión; debe abrir una nueva ventana. 

4. A continuación, puede cambiar el tamaño de las ventanas para corregirlas en paralelo, o alternar entre las vistas para comparar. 

5. Después de familiarizarse con la organización y el diseño de la experiencia experimental, haga clic en el botón **Encuesta del experimento** para enviar sus comentarios. Envíe el formulario antes de medianoche del 4 de abril de 2018 para que podamos revisar e incorporar los comentarios.
