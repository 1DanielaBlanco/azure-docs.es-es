# <a name="sql-data-warehouse-data-science-walkthroughs-using-t-sql-and-python-on-azure"></a>Tutoriales de ciencia de datos de SQL Data Warehouse mediante T-SQL y Python en Azure

Para obtener información general del proceso de ciencia de datos en equipo, consulte [Proceso de ciencia de datos](data-science-process-overview.md). Para consultar una introducción a SQL Data Warehouse, consulte [Azure SQL Data Warehouse](../sql-data-warehouse/sql-data-warehouse-overview-what-is.md)

## <a name="predict-taxi-tips-using-t-sql-and-ipython-notebooks-with-sql-data-warehouse"></a>Predicción de propinas para taxis mediante T-SQL y cuadernos de IPython con SQL Data Warehouse

En el tutorial sobre el [uso de SQL Data Warehouse](machine-learning-data-science-process-sqldw-walkthrough.md) se muestra cómo crear e implementar modelos de clasificación y regresión de aprendizaje automático mediante SQL Data Warehouse (SQL DW) para un conjunto de datos de carreras y tarifas de taxi de la ciudad de Nueva York disponible públicamente.
