# API de REST de Power BI Embedded

Con **Microsoft Power BI Embedded**, puede utilizar una API de REST con el fin de desarrollar mediante programación aplicaciones que integren los **informes de Power BI** en sus aplicaciones.

Para obtener información sobre la API de REST de **Power BI Embedded**, consulte las [operaciones de REST de Power BI en esta página](https://msdn.microsoft.com/library/mt711493).
