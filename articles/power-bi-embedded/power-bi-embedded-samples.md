# Muestras

[Descargar la muestra de versión preliminar pública de Power BI Embedded](http://go.microsoft.com/fwlink/?LinkID=761493)

Para ayudarle a empezar, hemos creado una aplicación web de ejemplo. Puede descargarla y leer el artículo complementario [Get started with the sample](power-bi-embedded-get-started-sample.md) (Introducción a la muestra). La muestra ilustra cómo realizar las siguientes operaciones:

  -	Aprovisionar contenido
      - Crear un área de trabajo
      - Importar un archivo PBIX
      - Actualizar las cadenas de conexión y establecer las credenciales para los conjuntos de datos

  - Insertar un informe de forma segura
