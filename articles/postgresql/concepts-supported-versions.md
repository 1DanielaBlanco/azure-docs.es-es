# <a name="supported-postgresql-database-versions"></a>Versiones de base de datos de PostgreSQL admitidas
Azure Database for PostgreSQL actualmente admite las siguientes versiones:

## <a name="postgresql-version-962"></a>Versión PostgreSQL 9.6.2
Consulte la [documentación de PostgreSQL](https://www.postgresql.org/docs/9.6/static/release-9-6-2.html) para obtener más información sobre las mejoras y correcciones de PostgreSQL 9.6.2.

## <a name="postgresql-version-956"></a>Versión PostgreSQL 9.5.6
Consulte la [documentación de PostgreSQL](https://www.postgresql.org/docs/9.6/static/release-9-5-6.html) para obtener información sobre las mejoras y correcciones de PostgreSQL 9.5.6.

## <a name="managing-updates-and-upgrades"></a>Administración de actualizaciones
Azure Database for PostgreSQL administra automáticamente la aplicación de revisiones para las actualizaciones de versión secundaria. Actualmente, en la versión preliminar pública, no se admite la actualización de versión principal. Por ejemplo, no se admite la actualización de PostgreSQL 9.5 a PostgreSQL 9.6.
