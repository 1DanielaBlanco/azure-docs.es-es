# <a name="azure-remoteapp-retirement"></a>Retirada de Azure RemoteApp
Se le dirigió a este artículo porque no nos ha sido posible ponernos en contacto con usted para confirmar la retirada de Azure RemoteApp. 

Estamos intentando ponernos en contacto con usted y los demás administradores de suscripción de Azure para comprobar que saben que Azure RemoteApp se retirará el 31 de agosto de 2017, tal como lo [anunciamos el 12 de agosto de 2016](http://aka.ms/araretirement).   Le solicitamos que responda de manera urgente a [arainfo@microsoft.com](mailto:arainfo@microsoft.com) indicando lo siguiente:
1.  Tanto usted como su organización confirman la retirada el 31 de agosto de 2017
2.  Cuéntenos sobre sus planes de migración, para ayudarlo si es necesario. Las opciones de migración se detallan en [Opciones para realizar una migración desde Azure RemoteApp](http://aka.ms/aramigration), como Citrix XenApp Essentials, MyCloudIT, Frame, etc. 

Es posible que el nodo REMOTEAPP dentro del portal de administración de Azure esté bloqueado. Esto se debe a que no nos pudimos poner en contacto con usted y fue necesario forzar que se comunicara con nosotros.  Es posible que no haya recibido correos electrónicos nuestros porque la dirección de correo electrónico asociada con la suscripción de Azure no se supervisa o no es válida.  Se recomienda encarecidamente validar que las direcciones de correo electrónico asociadas con la suscripción de Azure son válidas y se les supervisa. Para ello, vaya a Azure Portal y compruebe los administradores de suscripción.  

¿Tiene más preguntas o necesita ayuda para realizar la migración?  Escríbanos a [arainfo@microsoft.com](mailto:arainfo@microsoft.com) o [convoque a una reunión](http://aka.ms/ericorman) ahora mismo con el equipo encargado de la transición.
