# Introducción a la configuración avanzada de Administración de API de Azure

Los tutoriales de esta sección ofrecen orientación sobre tareas de configuración y administración avanzadas de Administración de API.

-   [Personalización del portal de desarrolladores en Administración de API de Azure][] - En este tutorial se muestra cómo modificar la apariencia del portal de desarrolladores a fin de unificarlo con su imagen de marca.
-   [Uso del Inspector de API para hacer un seguimiento de las llamadas en Administración de API de Azure][] - En este tutorial se muestra cómo utilizar el Inspector de API para realizar un seguimiento de las llamadas de operación y recopilar información sobre las mismas.
-   [Creación y ajuste de una configuración de producto avanzada en Administración de API de Azure][] - En este tutorial se muestra cómo configurar algunos ajustes avanzados de productos, incluyendo el límite de tasas y las directivas de cuotas y, a continuación, se ofrece una demostración de invocación de operaciones hasta que se supera el límite de tasas.
-   [Delegación de registros de usuario y suscripciones de producto][] - En este tutorial se muestra cómo utilizar su sitio web actual para controlar los registros e inicios de sesión de los desarrolladores y sus suscripciones a productos en contraposición al uso de la funcionalidad integrada en el portal de desarrolladores.
