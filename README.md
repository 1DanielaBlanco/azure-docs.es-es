# <a name="microsoft-azure-documentation"></a>Documentación de Microsoft Azure

> Entre el 5 y el 9 de marzo de 2019, se va a realizar la migración del repositorio MicrosoftDocs/azure-docs desde el motor de Markdown actual al motor de Markdown denominado [Markdig](https://github.com/lunet-io/markdig). Durante este tiempo vamos a suspender la capacidad de combinar las solicitudes de incorporación de cambios públicas para completar la migración.

Bienvenido a la [documentación](https://docs.microsoft.com/azure) de código abierto de [Microsoft Azure](https://azure.microsoft.com). Repase este archivo Léame para entender cómo puede ayudar a colaborar en la documentación de Microsoft Azure.

## <a name="getting-started"></a>Introducción

La colaboración con el código abierto es algo más que proporcionar actualizaciones. También es avisarnos cuando se produce un problema. Lea la [Guía de colaboración](.github/CONTRIBUTING.md) para más información.

### <a name="prerequisites"></a>Requisitos previos

Estupendo, ha decidido colaborar. Para colaborar a la documentación, necesita algunas herramientas.

La colaboración con la documentación requiere una cuenta de GitHub. Si no tiene una cuenta, siga las instrucciones para la [configuración de la cuenta de GitHub](https://docs.microsoft.com/contribute/get-started-setup-github) de la guía para colaboradores.

#### <a name="download"></a>Descargar

Instale las herramientas siguientes:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* Extensión de [Docs Authoring Pack](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) para Visual Studio Code

#### <a name="install"></a>Instalación

Siga las instrucciones que se proporcionan en [Instalación de herramientas de creación de contenido](https://docs.microsoft.com/contribute/get-started-setup-tools) de la guía para colaboradores.
