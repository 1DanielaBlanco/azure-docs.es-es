Para acceder a Azure Storage, necesitará una suscripción de Azure. Si todavía no tiene una suscripción, cree una [cuenta gratuita](https://azure.microsoft.com/free/?WT.mc_id=A261C142F) antes de empezar.

Todo el acceso a Azure Storage tiene lugar mediante una cuenta de almacenamiento. Si aún no tiene una cuenta de almacenamiento, puede crear una fácilmente mediante [Azure Portal](https://portal.azure.com/#create/Microsoft.StorageAccount-ARM). También puede crear una cuenta de almacenamiento con Azure PowerShell o la CLI de Azure. Si necesita ayuda para crearla, consulte [Creación de una cuenta de almacenamiento](../articles/storage/common/storage-quickstart-create-account.md).
