Para enviar el paquete al centro de datos de Azure se puede usar FedEx, UPS o DHL. 

- Debe proporcionar un número de cuenta válido de las empresas de transporte FedEx, UPS o DHL para que Microsoft realice la devolución. 
    
    - En el caso de FedEx, UPS o DHL, se requiere un número de cuenta para devolver las unidades enviadas desde Estados Unidos y Europa. 
    - En DHL, se prefiere un número de cuenta para devolver las unidades enviadas desde Asia y Australia. 
    - Si no tiene un número de cuenta, cree una cuenta de transportista [FedEx](http://www.fedex.com/us/oadr/) o [DHL](http://www.dhl.com/).
- Al enviar los paquetes, debe seguir los términos establecidos en los [Términos de servicio de Microsoft Azure](https://azure.microsoft.com/support/legal/services-terms/).
