## <a name="create-an-azure-sql-database-logical-server"></a>Creación de un servidor lógico de Base de datos SQL de Azure
Utilice los pasos siguientes en el Portal de Azure para crear un servidor lógico de Base de datos SQL de Azure.

1. Si no está conectado, conéctese a [Azure Portal](http://portal.azure.com).
2. Haga clic en **Nuevo**, haga clic en **Datos y almacenamient** y busque **SQL Database**.
3. Haga clic en **Base de datos SQL** para abrir la hoja Base de datos SQL.
4. Proporcione los valores para las propiedades de servidor siguientes:
   
   * Nombre de servidor
   * Inicio de sesión de administrador de servidor
   * Password
   * Suscripción (solo si tiene varias suscripciones)
   * Grupo de recursos (nuevo o uno existente)
   * Ubicación
5. Haga clic en **Crear**. En el área de notificación, verá que la implementación se ha iniciado.
6. Espere a que la implementación finalice antes de continuar con el paso siguiente.
