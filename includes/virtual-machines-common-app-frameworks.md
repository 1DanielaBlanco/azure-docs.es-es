## <a name="azure-cli-20"></a>CLI de Azure 2.0
Estos pasos requieren la CLI de Azure versión 2.0.4 o versiones posteriores. Ejecute `az --version` para encontrar la versión. Si necesita actualizarla, consulte [Instalación de la CLI de Azure 2.0]( /cli/azure/install-azure-cli).

En primer lugar, cree un grupo de recursos con [az group create](/cli/azure/group#create). A continuación, cree la implementación desde una plantilla con [az group deployment create](/cli/azure/group/deployment#create). Proporcione el identificador URI completo a la plantilla con el parámetro `--template-uri` y los parámetros necesarios de la plantilla en `--parameters` como se indica a continuación:

```azurecli
az group create --name myResourceGroup --location eastus
az group deployment create --resource-group myResourceGroup \
    --template-uri https://raw.githubusercontent.com/azure/azure-quickstart-templates/master/{template-name}/azuredeploy.json \
    --parameters '{"sshKeyData": {"value": "ssh-rsa AAAAB3N{snip}B9eIgoZ"}}'
```

Dependiendo de la plantilla, es posible que Azure tarde algún tiempo en implementar los recursos.


## <a name="azure-powershell"></a>Azure PowerShell
Estos pasos requieren la versión 3.6 o posterior del módulo Azure PowerShell. Ejecute `Get-Module -ListAvailable AzureRM` para encontrar la versión. Si necesita actualizarla, consulte [Instalación del módulo de Azure PowerShell](/powershell/azure/install-azurerm-ps).

En primer lugar, cree un grupo de recursos con [New-AzureRmResourceGroup](/powershell/module/azurerm.resources/new-azurermresourcegroup). A continuación, cree la implementación desde una plantilla con [AzNew-AzureRmResourceGroupDeployment](/powershell/module/azurerm.resources/new-azurermresourcegroupdeployment). Proporcione el identificador URI completo a la plantilla con el parámetro `-TemplateURI` como sigue:

```powershell
New-AzureRmResourceGroup -Name "myResourceGroup" -Location "EastUS"
New-AzureRmResourceGroupDeployment -ResourceGroupName "myResourceGroup" `
     -TemplateUri "https://raw.githubusercontent.com/azure/azure-quickstart-templates/master/{template-name}/azuredeploy.json"
```

Al ejecutar el comando [New-AzureRmResourceGroupDeployment](/powershell/module/azurerm.resources/new-azurermresourcegroupdeployment), se le pedirá que escriba los valores de los parámetros en la plantilla. Dependiendo de la plantilla, es posible que Azure tarde algún tiempo en implementar los recursos.
