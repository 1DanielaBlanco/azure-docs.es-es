## <a name="a-namesetupdevenvaset-up-the-development-environment"></a><a name="setupdevenv"></a>Configuración del entorno de desarrollo
Para comenzar, configure su entorno de desarrollo mediante la instalación del [SDK de Azure para Visual Studio 2013][Azure SDK for Visual Studio 2013].

Si no cuenta con Visual Studio instalado, se instalará Visual Studio Express para Web junto con el SDK.

> [!NOTE]
> Según la cantidad de dependencias de SDK que tenga ya en la máquina, la instalación del SDK puede tardar un período largo, desde unos minutos a media hora o más.
> 
> 

[Azure SDK for Visual Studio 2013]: http://go.microsoft.com/fwlink/?LinkID=324322
