Tenga en cuenta las siguientes consideraciones de seguridad al usar StorSimple Cloud Appliance:

* El dispositivo de nube está protegido mediante su suscripción de Microsoft Azure. Esto significa que si usa el dispositivo de nube y su suscripción de Azure se pone en peligro, los datos almacenados en él también están en peligro.
* La clave pública del certificado usada para cifrar los datos almacenados en StorSimple está disponible de forma segura en el portal de Azure; la clave privada se conserva con StorSimple Cloud Appliance. En StorSimple Cloud Appliance, tanto las claves públicas como las privadas se almacenan en Azure.
* El dispositivo de nube está hospedado en el centro de datos de Microsoft Azure.
