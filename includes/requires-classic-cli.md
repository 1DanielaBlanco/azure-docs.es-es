> [!IMPORTANT]
> En este artículo se incluye contenido que requiere la CLI de Azure clásica. La versión actual de la CLI de Azure no es compatible con las características que se describen en este artículo, por lo que se requiere la CLI clásica. 
> 
> La CLI clásica se puede instalar junto con la CLI de Azure moderna, pero recomendamos usar la CLI de Azure para todos los nuevos scripts e implementaciones en los que se disponga de compatibilidad de características. Para instalar la CLI clásica, consulte [Instalación de la CLI clásica de Azure](https://docs.microsoft.com/cli/azure/install-classic-cli?view=azure-cli-latest). 
> Para instalar la CLI actual, consulte [Instalación de la CLI de Azure](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest).
