> [!TIP]
> 
> Administre recursos de Azure Blob Storage con el Explorador de Azure Storage. 
> El [Explorador de Azure Storage](https://azure.microsoft.com/features/storage-explorer/) es una aplicación independiente y gratuita de Microsoft que permite [administrar recursos de Azure Blob Storage](../articles/vs-azure-tools-storage-explorer-blobs.md). Mediante el Explorador de Azure Storage, puede crear visualmente, leer, actualizar y eliminar blobs y contenedores de blobs, así como administrar el acceso a los suyos.
