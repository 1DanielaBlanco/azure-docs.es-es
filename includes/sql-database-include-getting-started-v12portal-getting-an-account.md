## <a name="connecting-to-the-azure-portal-with-a-subscription"></a>Conexión al Portal de Azure con una suscripción
Si tiene una suscripción a Microsoft Azure, siga estos pasos para conectarse al Portal de Azure. En caso contrario, haga clic en uno de los siguientes vínculos para obtener una cuenta de Microsoft Azure y, después, siga los pasos que figuran a continuación para conectarse al Portal de Azure.

1. Abra el explorador que prefiera y conéctese a [Azure Portal](https://portal.azure.com/) en http://portal.azure.com/.
2. Inicie sesión en [Azure Portal](https://portal.azure.com/) en http://portal.azure.com/.
3. Cuando aparezca la página de inicio de sesión, proporcione las credenciales de la suscripción.
