> [!IMPORTANT]
> En este artículo incluye contenido que requiere la CLI de Azure clásica. Es una versión anterior de la CLI de Azure que debe usarse solo con el modelo de implementación clásica.
>
> La CLI clásica se puede instalar junto con la CLI Azure moderna, pero recomendamos usar la CLI Azure para todos los nuevos scripts e implementaciones.
> Para instalar la CLI clásica, consulte [Instalación de la CLI de Azure clásica](/cli/azure/install-classic-cli) y para instalar la CLI actual, consulte [Instalación de la CLI de Azure](/cli/azure/install-azure-cli).
