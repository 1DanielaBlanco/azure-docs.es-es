> [!IMPORTANT]
> Esta característica se encuentra en su versión preliminar. Para usarla, debe instalar la [extensión de Event Grid](/cli/azure/azure-cli-extensions-list) para [CLI de Azure](/cli/azure/install-azure-cli). Puede instalarla con `az extension add --name eventgrid`. Si instaló la extensión anteriormente, actualícela con `az extension update -n eventgrid`. Si usa la API de REST, asegúrese de utilizar `api-version=2018-09-15-preview`.
