Para crear y administrar un clúster de informática de alto rendimiento (HPC) basado en la nube, aproveche los servicios de infraestructura y proceso de Azure y [Microsoft HPC Pack](https://technet.microsoft.com/library/jj899572.aspx) . HPC Pack, que se puede descargar de forma gratuita, se basa en las tecnologías de Microsoft Azure y Windows Server, y admite una amplia variedad cargas de trabajo de HPC.

Para obtener información acerca de otras opciones de HPC, consulte [Recursos técnicos para informática de alto rendimiento y computación por lotes](../articles/batch/big-compute-resources.md).
