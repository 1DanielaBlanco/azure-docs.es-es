> [!NOTE]
>
> Este artículo se actualizó recientemente para usar comandos del nuevo módulo Az de Azure PowerShell. Se trata del nuevo módulo de PowerShell para Azure, compatible con las versiones 5.x y 6 de PowerShell. AzureRM seguirá recibiendo actualizaciones de correcciones críticas, pero las características nuevas solo estarán en el módulo Az.
>
> * Para más información sobre el módulo AZ y el soporte técnico continuado para el módulo AzureRM, consulte [Presentación del nuevo módulo Az de Azure PowerShell](/powershell/azure/new-azureps-module-az).
> * El módulo Az ya está disponible en Cloud Shell, con la compatibilidad con AzureRM habilitada de manera automática.
>   Para instrucciones sobre cómo instalar el módulo Az, consulte [Instalar Azure PowerShell](/powershell/azure/install-az-ps).
> * Para habilitar los comandos de compatibilidad de AzureRM, ejecute `Enable-AzureRmAlias` después de la instalación.
>   Para más información sobre la compatibilidad, consulte el artículo sobre la [migración desde AzureRM de Azure PowerShell a Az](/powershell/azure/migrate-from-azurerm-to-az).
