# Bibliotecas y herramientas de desarrollo de Azure por lotes 
<p> Obtenga estas bibliotecas y herramientas para desarrollar aplicaciones por lotes y lotes de Azure.

+ [Lote]
+ [Aplicaciones por lotes]
+ [Recursos adicionales]

## Lote
 
+ [Biblioteca de cliente de procesamiento por lotes para .NET](http://www.nuget.org/packages/Azure.Batch/) (NuGet): desarrolle aplicaciones cliente para ejecutar trabajos con el servicio de proceso por lotes
+ [Biblioteca de administración por lotes](http://www.nuget.org/packages/Microsoft.Azure.Management.Batch/) (NuGet): cuentas de servicio de administración por lotes
+ [Explorador de lotes](https://code.msdn.microsoft.com/windowsazure/Azure-Batch-Explorer-c1d37768) (MSDN): cree una aplicación de interfaz gráfica de usuario para ver y administrar trabajos por lotes

## Aplicaciones por lotes

+ [SDK de nube de aplicaciones por lotes](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps.Cloud/) (NuGet): permiten a las aplicaciones descargar trabajos al servicio de proceso por lotes
+ [Extensión de aplicaciones por lotes de Visual Studio](https://visualstudiogallery.msdn.microsoft.com/8b294850-a0a5-43b0-acde-57a07f17826a) (Galería de Visual Studio): habilite aplicaciones para la nube en Visual Studio mediante el SDK de nube de aplicaciones por lote
+ [SDK de cliente de aplicaciones por lotes](http://www.nuget.org/packages/Microsoft.Azure.Batch.Apps/) (NuGet): interactúe con aplicaciones habilitadas para descargar al servicio de proceso por lotes
